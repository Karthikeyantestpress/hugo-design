# 🚀 Hugo Design Starter

## What is this?

* A ready-to-use **Hugo starter** with Tailwind v4, Preline UI, and Alpine
* Helps you quickly build **modern dashboards and web UI**

---

## ⚙️ Setup (Mac & Linux)

### 1. Install Hugo

#### 🍏 macOS

```bash
brew install hugo
hugo version
```

#### 🐧 Linux / WSL (Recommended)

```bash
wget https://github.com/gohugoio/hugo/releases/latest/download/hugo_extended_linux-amd64.deb
sudo dpkg -i hugo_extended_*.deb
sudo apt-get install -f -y
hugo version
```

---

### 2. Clone the repository

```bash
git clone git@github.com:Karthikeyantestpress/hugo-design.git
cd hugo-design
```

---

### 3. Install dependencies

```bash
npm install
```

---

### 4. Run the project

```bash
hugo server -D
```

---

### 5. Open in browser

```
http://localhost:1313
```
